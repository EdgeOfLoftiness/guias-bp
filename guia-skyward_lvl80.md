# BPSR Skyward – Build lvl 80 con lanza lvl 70 fija (Crit ~30%, Haste ~25%)

Este documento resume la distribución de estadísticas por pieza siguiendo las reglas finales del set y la **lanza lvl 70 obligatoria**.  
Objetivo: llegar a **threshold de Crit (~30%)**, luego a **Haste ~25%**, y el resto en **Mastery / Versatility**.

## Conversiones usadas
A partir de tus valores reales:

### Crit
- `2668 Crit = 37.44%`
- Conversión:
  - `1% Crit ≈ 2668 / 37.44 ≈ 71.26 Crit`
  - `200 Crit ≈ 200 / 71.26 ≈ 2.81% Crit`

### Haste
- `429 Haste = 8.78%`
- Conversión:
  - `1% Haste ≈ 429 / 8.78 ≈ 48.88 Haste`
  - `200 Haste ≈ 200 / 48.88 ≈ 4.09% Haste`

### Base
- `Crit base del personaje = 5%`
- Todo lo demás parte desde `0`.

---

## Razonamiento de la build

### 1) Crit target ~30%
Crit requerido desde 0:

- Crit total necesario para 30% con base 5%:
  - Necesitas `25%` desde gear.
  - `25 × 71.26 ≈ 1781.5 Crit pts`.

Crit fijo que ya viene sin elegir nada:
- De 3 piezas set obligatorias: `3 × 200 = 600 Crit`
- De la lanza lvl 70: `+306 Crit`
- Total fijo inicial: `906 Crit`.
- Eso equivale a `906 / 71.26 ≈ 12.71%` desde gear.  
  Con base 5% → `≈17.7% Crit` al empezar.

Crit que falta para llegar a 30%:
- `1781.5 - 906 ≈ 875.5 Crit`.

Con la regla de no repetir main/sub en piezas libres:
- Máximo eficiente por pieza libre para una stat prioritaria:
  - `Main X 200 + Roll X 60 = 260`.
  - (Porque Sub no puede ser X si Main es X).

Entonces para cubrir ~875 Crit faltante:
- `875.5 / 260 ≈ 3.36` → se resuelve con:
  - la 4ª pieza set (si la eliges) aporta `+200 Crit`
  - 2 piezas libres con Main Crit (+260 cada una con roll Crit)
  - rolls adicionales a Crit en otras piezas.
  
Resultado esperado final: `Crit ~30–32%` (margen aceptable).

### 2) Haste target ~25%
Haste requerido desde 0:

- `25% × 48.88 ≈ 1222 Haste pts`.

Haste se obtiene de:
- Main Haste en varias piezas libres.
- Sub Haste solo donde no choque con Main.
- Rolls Haste.

Cada pieza libre con prioridad Haste puede aportar:
- `Main Haste 200 + Roll Haste 60 = 260`
- y si el Sub es Haste (solo cuando Main no es Haste): `+100` extra.

Con 4 piezas Main Haste + algunos Subs Haste + 4 rolls a Haste:
- se llega a `~25%` con margen leve (25–26%).

### 3) Lo que sobra
Una vez alcanzados los thresholds:
- Preferir **Mastery** y **Versatility** en Subs libres.
- Haste pierde prioridad después del threshold.
- Luck se ignora salvo lo fijo del set/arma.
